## Environment Variables to Set

## Deployment Steps

1. [ ] Run `bash setup-huggingface.sh`
2. [ ] Create new Docker Space on Hugging Face
3. [ ] Clone the Space repository
4. [ ] Copy all OpenClaw files to Space directory
5. [ ] Commit and push to Hugging Face
6. [ ] Set environment variables in Space settings
7. [ ] Wait for build to complete (~5-10 minutes)
8. [ ] Access your gateway at the Space URL
9. [ ] Configure AI models in web UI
10. [ ] Test the gateway

## Common Issues

### ‚ùå "Missing configuration in README"
- **Fix**: Ensure README.md has YAML frontmatter (lines 1-11)
- **Verify**: First line should be `---`

### ‚ùå "Port not accessible"
- **Fix**: Use `Dockerfile.huggingface` with port 7860
- **Verify**: CMD line includes `--port 7860`

### ‚ùå "Build fails"
- **Fix**: Check build logs for specific errors
- **Common**: Missing files, wrong Node version
- **Solution**: Ensure all files copied, Dockerfile uses Node 22

### ‚ùå "Gateway not responding"
- **Fix**: Check Space is running (not sleeping)
- **Verify**: Environment variables are set
- **Check**: Build completed successfully
